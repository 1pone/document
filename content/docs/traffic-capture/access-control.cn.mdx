---
title: 应用权限管理
---

创建应用的用户即为该应用的 Owner，拥有所有操作权限，普通用户仅可启动回放、查看回放报告等。

|    | Owner 操作权限 | 普通用户操作权限 |
| ---------------- | -------- | -------- |
| 修改配置         | √        | ×        |
| 启动回放         | √        | √        |
| 查看回放统计信息 | √        | √        |
| 删除应用         | √        | ×        |
| 查看录制详情（脱敏前数据）     | √        | ×        |
| 添加其他成员为 Owner     | √        | ×        |

前往 **Report -> AppSetting -> Other** 进行设置，Owner 可以添加组织成员成为 Owner。
